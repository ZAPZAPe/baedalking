## 테스트 시나리오

### 신규 사용자 플로우
1. 메인 페이지 접속
2. 회원가입 버튼 클릭
3. 이메일/비밀번호 입력
4. 프로필 설정 (닉네임, 지역, 차량)
5. 500P 획득 확인
6. 실적 업로드 시도
7. 100P 추가 획득 확인

### 기존 사용자 플로우
1. 로그인
2. 오늘 실적 업로드
3. 랭킹 확인
4. 포인트 내역 확인
5. 친구 초대

### 관리자 플로우
1. `/admin-login` 접속
2. 비밀번호 입력
3. 대시보드 확인
4. 사용자 관리
5. 배달 기록 조회
6. 로그아웃
